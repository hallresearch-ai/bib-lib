# bib-lib

## Usage

* Build venv using Python 3. 

* Open Pycharm project and run files from main. 

* Run `parser.py` to generate `.csv`s from existing `.bib`s.
  * Should be only `lib-YYYY-MM-DD-HHMM.bib`, but can be used to read multiple `.bib` files. 
  * If more than `.csv` is created, run `load_all_csv.py` to concat them into one `.csv`. Manually check if needed.  

* Run `write_all.py` to create a new master `lib.bib-YYYY-MM-DD-HHMM.bib`.
